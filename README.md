# Explore India - Travel & Tourism Website
## Features
* Home Page
* About Us Page
* Destinations Page
* Tour Packages Page
* Services Page
* Gallery Page
* Booking Form
* Contact Page
* Navigation Menu
* Customer Testimonials
* Frequently Asked Questions (FAQ)
* Travel Booking System
* Responsive and Structured Layout
---

* HTML5
* Semantic HTML Elements
* Forms and Input Controls
* Tables

* Hyperlinks
* Multimedia Integration

##  How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it using any web browser.
4. Navigate through the website using the menu links.
---
##  Website Pages
### Home Page
Provides an introduction to Explore India and highlights popular destinations.
### About Us
Contains information about the organization, mission, vision, achievements, and team members.
### Destinations
Displays famous tourist destinations across India.
### Tour Packages
Shows available travel packages and pricing details.
### Services
Lists travel-related services offered to customers.
### Gallery
Displays travel photographs and tourism attractions.
### Booking Page
Allows users to submit travel booking requests.
### Contact Page
Provides contact details and customer inquiry form.
